Telecommunications Optical Network System

This portfolio presents portions of a hardware installation guide and a system description guide for a telecommunications optical network system. The hardware installation guide is written for field installation engineers. The system description guide is written for planners, network operations personnel and system administrators who are responsible for deploying and administering digital optical networks.

I was responsible for updating these guides through several product releases, and worked as part of a full technical publications team. The documents were created using Adobe Framemaker. Some graphics were created utilizing an outside graphics design firm, while others were created using Microsoft Powerpoint and a corporate library of standardized images.

These documents are not public, so I cannot provide the guides in full. The samples below present only the table of contents and introductory material from each guide, which outline the scope of each document.
