Product Guides Portfolio

Enterprise Software

During my time working at an enterprise software firm, I orchestrated the development and release of a suite of product guides for a new vertical market product. In addition to personally writing two of the guides, I held responsibility for overseeing the team of technical writers required for the project; researching and defining the product documentation content, structure, and standard terminology; and maintaining documentation plans and schedules.

This portfolio presents portions of two guides that I personally wrote for the inaugural product release: an overview guide and installation guide. The overview guide is written for all users of the product. The installation guide is written for system administrators responsible for installing and setting up the software product. The documents were developed using Adobe Framemaker. Graphics were created using Microsoft Powerpoint and a screen capture tool.

These documents are not public, so I cannot provide the guides in full. The samples below present the table of contents from each guide, which outline the scope of each document.
